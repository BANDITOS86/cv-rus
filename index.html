Пример кода

Этот код определяет функцию, называемую greet. Когда функция вызывается, она предлагает пользователю ввести свое имя с помощью всплывающего окна и сохраняет введенные данные в переменной name. Затем создаётся новый объект Date и присваивается его переменной date для представления текущей даты и времени. Метод toLocaleDateString() вызывается для объекта date для преобразования даты в строковое представление на основе языкового стандарта пользователя. Результирующая строка сохраняется в переменной today. Наконец, отображается сообщение с использованием литералов шаблона, которое включает имя пользователя и отформатированную текущую дату. 

            
              function greet() {
                const name = prompt('Enter your name:');
                const date = new Date();
                const today = date.toLocaleDateString();
                alert(`Hello, ${name}! Today is ${today}.`);
              }
              
              greet();
